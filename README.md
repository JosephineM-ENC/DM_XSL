# *L'Albatros*, une métaphore du poète dans le recueil *Les Fleurs du Mal* de Baudelaire

Bienvenu dans le dépôt consacré à mon rendu de projet final du cours "Techniques et chaîne de publication électronique" de M2 Technologies numériques appliquées à l'histoire (TNAH) de École nationale des chartes, 2025-2026. 

Le rendu attendu par le professeur est un projet indivuel présentant une feuille de style XSL qui transforme un document XML. Pour cela, l'étudiant doit rendre un dépôt GitHub (que vous lisez) composé du document XML (la source albatros_tei.xml), de la feuille de style XSL (transformation.xsl) ainsi que de la sortie des fichiers HTML générés (le résultat index.html, poeme.html, analyse.html), présentée sous la forme d'un petit site. Vous pouvez consulter ce mini-site à cette adresse : https://josephinem-enc.github.io/DM_XSL/



# Explication du projet 

Mon projet consiste en la réalisation d'un petit site HTML d’analyse littéraire des *Fleurs du Mal* de Baudelaire, à travers l'exemple de *l'Albatros*. 
Ce site a été réalisé pour mettre en valeur une feuille de style XSL transformant un document XML (issu du projet du cours de TEI) encodant le poème *l'Albatros* de Charles Baudelaire. 
La transformation du document XML aura pour objectif de générer des pages HTML à l’aide d’une feuille XSLT afin de présenter le poème *l’Albatros*.

# *L'Albatros*, une métaphore du poète dans le recueil *Les Fleurs du Mal* de Baudelaire

Bienvenu dans le dépôt consacré à mon rendu de projet final du cours "Techniques et chaîne de publication électronique" de M2 Technologies numériques appliquées à l'histoire (TNAH) de École nationale des chartes, 2025-2026. 

Le rendu attendu par le professeur est un projet indivuel présentant une feuille de style XSL qui transforme un document XML. Pour cela, il est attendu de l'étudiant de rendre un dépôt GitHub composé du document XML (la source), de la feuille de style XSL (la transformation) ainsi que de la sortie des fichiers HTML générés (le résultat), présentée sous la forme d'un petit site.
